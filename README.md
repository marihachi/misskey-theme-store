# Misskey Theme Store
A theme store of Misskey.  

## Requirements
- MongoDB
- Node.js (with NPM)

## Usage
### 0. Installation
`npm i`

### 1. Build
for development:

`npm run build.dev`

for production:

`npm run build`

### 2. Generate server config
`npm run init`

### 3. start server
`npm run start`

Enjoy!
